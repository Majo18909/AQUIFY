# Carpeta de Imágenes - AQUIFY

## Logo de AQUIFY

Para agregar el logo de AQUIFY:

1. Guarda la imagen del logo como `aquify-logo.png` en esta carpeta
2. Formatos aceptados: PNG (preferido con fondo transparente), JPG, SVG
3. Tamaño recomendado: 200x200px o similar
4. La imagen se mostrará en el header de la aplicación

### Colores del logo:
- **Teal Primario**: #2B5F6F
- **Teal Oscuro**: #1E4451
- **Teal Medio**: #3A7A8C
- **Teal Claro**: #5A9BAE
- **Teal Muy Claro**: #A8D5E2
- **Blanco**: #FFFFFF

Estos colores se han integrado en la aplicación para mantener la identidad visual del logo.
